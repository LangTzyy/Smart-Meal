AR Technology

dalam Dunia Nyata

Teknologi Augmented Reality memungkinkan kamu melihat visualisasi 3D makanan secara langsung di kamera HP — sebelum kamu makan, kamu sudah tahu nutrisinya.

  • 📡

    Real-time AR Detection — Kamera otomatis mendeteksi makanan dan menampilkan overlay informasi nutrisi secara instan.

  • 📊

    Instant Nutrition Data — Kalori, protein, lemak, dan karbohidrat tampil langsung saat kamera diarahkan ke makanan.

  • 🧠

    AI-Powered Recognition — Didukung oleh sistem AI yang terintegrasi dengan API Anthropic Claude.

  • 🌐

    No App Install Needed — Berjalan langsung di browser menggunakan teknologi WebAR modern. Tidak perlu download apapun.
